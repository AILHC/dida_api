# 退出登录滴答清单

## WEB退出

> https://www.dida365.com/signout

*请求方式：GET*

请求头需要带上的cookie

| 字段       | 值                | 必要性 | 作用       |
| ---------- | ----------------- | ------ |----------|
| t          | 登录成功后的token | 必要   |          |
| AWSALB     | 不明              |        | 亚马逊的负载均衡(无用)  |
| AWSALBCORS | 不明              |        | 亚马逊的负载均衡(无用) |

**退出成功**

| 状态码 | 响应体                  |
| ------ | ----------------------- |
| 302    | Found. Redirecting to / |
